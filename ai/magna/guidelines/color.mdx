> **Who this is for:** Engineers building learner-facing UI in `gl-app-native` with `@gl/elements`.
> **What this covers:** When to reach for which color role, how M3 pairing keeps contrast safe by default, and how to express elevation tonally.

## Why it matters

Magna is the learner surface. Learners decide in seconds whether they trust a screen — colors that *look* like buttons, alerts, or progress carry meaning before the copy is read. M3 role tokens are the only way that meaning stays consistent across light and dark, across screens, and across team members.

## Core principles

1. **Use role tokens, never raw hex.** Hex disappears the moment a designer ships a new palette; the role survives.
2. **Pair roles.** Text on `$primary` is `$onPrimary`. Text on `$surfaceContainer` is `$onSurface`. Pairing is the accessibility rule.
3. **Express elevation tonally.** Step up `$surfaceContainer` → `$surfaceContainerHigh` instead of adding shadows.
4. **Status colors come from extended `magnaColors`.** Success/warning/info already ship with container + on-container pairs — don't invent loose greens or ambers.
5. **Light/dark flips for free.** Reference the role; the theme handles the mode.

## How to implement

### Surfaces and text
```tsx
import { Surface, Typography } from '@gl/elements';

<Surface backgroundColor="$surfaceContainer" padding="$md">
  <Typography variant="subtitle1" color="$onSurface">Course title</Typography>
  <Typography variant="body2" color="$onSurfaceVariant">12 lessons · 4h 30m</Typography>
</Surface>
```

### Interactive (button)
```tsx
import { Button } from '@gl/elements';

// $primary fill + $onPrimary text are wired by the component
<Button onPress={enroll}>Enroll</Button>
```

### Status (use the extended role)
```tsx
import { Surface, Typography } from '@gl/elements';

<Surface backgroundColor="$successContainer" padding="$md" borderRadius="$lg">
  <Typography color="$onSuccessContainer">Lesson 4 complete</Typography>
</Surface>
```

## Do / Don't

| Do | Don't |
|---|---|
| `color="$onSurface"` for body text | `color="#1A1B1E"` |
| `backgroundColor="$primaryContainer"` for tonal fills | `backgroundColor="lightblue"` |
| Raise cards by stepping `$surfaceContainer*` | Add a `boxShadow` to highlight a card |
| `<Typography color="$onPrimary">` inside a `$primary` button | `<Typography color="$onSurface">` inside a `$primary` button |

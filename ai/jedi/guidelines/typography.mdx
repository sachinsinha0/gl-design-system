> **Who this is for:** Engineers picking type styles for internal / partner / admin surfaces with `@gl/jedi`.
> **What this covers:** The Inter scale, why headings are 600 with tight tracking, and the mobile rules that diverge from Magna.

## Why it matters

Hierarchy in Jedi comes from **weight and size**, not color — headings are the same ink as body. That discipline is what lets information-dense screens stay scannable without screaming. The 600-weight, -0.4px-tracking headline is the recognizable Jedi shape; mess with it and the system stops looking like itself.

## Core principles

1. **Single family: Inter.** Weights 400 / 500 / 600 — load only those three.
2. **Variants are closed.** `h1`–`h5` (no `h6`), `subtitle1`/`2`, `body1`/`2`, `caption`, `overline`, `button`. A sixth heading level = the hierarchy is too deep.
3. **Headings 600 weight with -0.4px letter-spacing.** Compact, modern, product-y. Keep headlines short.
4. **UPPERCASE only on buttons and overlines.** Everything else is sentence/title case.
5. **Mobile body shrinks** (unlike Magna): below `md`, `body1` → 14/20, `body2` → 12/16, letter-spacing relaxes to 0.
6. **Dark mode flips Button weight to 600 automatically.** Don't override.

## How to implement

### Headings + body + label
```tsx
import { Typography, Stack } from '@mui/material';

<Stack spacing={2}>
  <Typography variant="h2">Cohort overview</Typography>
  <Typography variant="subtitle1">January 2026 batch</Typography>
  <Typography variant="body1">42 learners, 3 program managers, 9 courses.</Typography>
</Stack>
```

### Section heading (closed scale)
```tsx
<Typography variant="h5">Deepest heading level</Typography>  {/* ✅ */}
<Typography variant="h6">Deepest heading level</Typography>  {/* ❌ doesn't exist */}
```

### Don't reintroduce the family
```tsx
<Typography sx={{ fontFamily: 'Roboto' }}>…</Typography>     {/* ❌ */}
<Typography>…</Typography>                                   {/* ✅ inherits Inter */}
```

## Do / Don't

| Do | Don't |
|---|---|
| `<Typography variant="h5">` for the deepest heading | `<Typography variant="h6">` (doesn't exist) |
| Use `subtitle1` / `subtitle2` for labels | Use `body1` bolded as a label |
| Stick to Inter | Set `fontFamily: 'Roboto'` |
| UPPERCASE only on buttons + overline | UPPERCASE headings or chips |

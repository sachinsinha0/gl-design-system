> **Who this is for:** Anyone authoring or styling marketing copy — engineers, content editors, designers shipping new landing pages.
> **What this covers:** The Poppins scale, the discrete mobile/desktop swap at 768px (not fluid `clamp()`), and the UPPERCASE-only-on-buttons rule.

## Why it matters

Marketing copy lives on every kind of device, browser, and ad-network embed. A fluid `clamp()` scale looks elegant in a designer's browser and falls apart in a Facebook in-app webview. GLDS-Web ships two **discrete** scales that **swap** at 768px — predictable, debuggable, copy-pastable into any stack.

## Core principles

1. **Poppins for all new work**, weights 400 / 500 / 600. Legacy is Poppins headings + Noto Sans body — migrate to the Web scale when touching old templates.
2. **Type swaps at 768px** (`--glds-breakpoint-md`). It does **not** fluidly interpolate. `1024px` governs layout (columns, max-widths), not type.
3. **Keep headlines short.** H1 must survive 40px on mobile.
4. **UPPERCASE only on Button and Overline.** Headlines and body are sentence/title case.

## How to implement

### Desktop scale (Poppins)

| Style | Size | Weight | Line height | Letter spacing | Case |
|---|---|---|---|---|---|
| Headline 1 | 64px | 600 | 80px | -2% | — |
| Headline 2 | 48px | 600 | 64px | -2% | — |
| Headline 3 | 32px | 600 | 40px | 0 | — |
| Body 1 | 16px | 400 | 24px | 0 | — |
| Button | 16px | 600 | 16px | 0.5px | UPPERCASE |
| Overline | 12px | 600 | 16px | 20% | UPPERCASE |

### Responsive swap (mobile-first)
```css
.glds-hero__title {
  font-family: var(--glds-font-family);
  font-size: 40px;
  line-height: 48px;
  font-weight: 600;
  letter-spacing: -0.02em;
}
@media (min-width: 768px) {
  .glds-hero__title {
    font-size: 64px;
    line-height: 80px;
  }
}
```

### Don't reintroduce the family
```css
.headline { font-family: 'Roboto'; }                /* ❌ */
.headline { font-family: var(--glds-font-family); } /* ✅ */
```

## Do / Don't

| Do | Don't |
|---|---|
| `font-family: var(--glds-font-family)` (Poppins) | `font-family: 'Roboto'` (or Noto Sans for new work) |
| Use the responsive pair for headlines | One giant `clamp()` per heading |
| UPPERCASE only on Button + Overline | UPPERCASE the headline for emphasis |
| Keep H1 short — survives 40px on mobile | A 12-word hero headline that wraps awkwardly on phones |
